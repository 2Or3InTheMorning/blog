## 稳定性
> 假定在待排序的记录序列中，存在**多个具有相同的关键字**的记录。若经过排序，这些记录的**相对次序保持不变**，即在原序列中，r[i]=r[j]，且r[i]在r[j]之前，而在排序后的序列中，r[i]仍在r[j]之前，则称这种排序算法是稳定的；否则称为不稳定的。

例如：待排序数组为[3, 1, 1, 2]，有两个相同关键字（1）。经过稳定算法排序后，原序列前方的1依旧在序列中还保持在前方，这就是稳定的。
