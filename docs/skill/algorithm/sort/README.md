## 稳定性
在待排序中，存在 **多条相同的记录**。如果排序后，这些相同记录的 **相对次序** 没有发生改变，则称这种排序算法是稳定的；

> 例如：待排序数组为[3, 1, 1, 2]，有两个相同关键字（1）。经过稳定算法排序后，原序列前方的1依旧在序列中还保持在前方，这就是稳定的。
