# CI/CD

![alt](./img/img-1.png)

## CI持续集成（Continuous Integration）
针对应用代码的新更改会定期构建、测试、合并到共享存储库。

## CD持续交付（Continuous Delivery）
确保尽可能减少部署新代码时所需的工作量

## CD持续部署（Continuous Deployment）
自动地将新更改从存储库发布到生产环境。
