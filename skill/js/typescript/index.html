# 如何理解TypeScript

TypeScript是JavaScript的超集，主要提供了类型系统、对ES6的支持。通过TypeScript静态类型检查，可以在开发时尽早发现语法错误或传参错误，也增强代码的阅读性、可维护性

# type与interface

相同点：

  • 都可以描述一个对象或函数的类型
  • 都可以 互相继承（继承的语法不同）

不同点：

  • type一般用来 声明类型别名、联合类型：
  • 在 type 语句中，可以通过typeof获取实例的类型，并赋值
  • interface可以 将同一类型的声明进行合并

# 示例

    name: string;
    age: number;
}
