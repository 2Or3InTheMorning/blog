# 动态规划

动态规划问题是递归问题查表，避免重复计算，从而节省时间。

# 大致思想

一般是利用递归，对每个元素 进行同样或类似的方法 进行计算。

一般是需要查找最大和、从后往前推等等

大致步骤：

  • 定义 状态数组dp[i] 的含义
  • 推出 状态方程
